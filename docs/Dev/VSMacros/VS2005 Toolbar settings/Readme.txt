Steps to restore the Toolbar settings.

1. Close all instances of IDEs.
2. Load the macros, used by toolbars.
	2.1 Merge the "VS 2005 LoadMacroProject RJK_InDn.reg"
3. Open VS 2005.
4. Import settings file "Exported-VS2005ToolbarSettings 2010-Apr-02.vssettings"
5. Close IDEs.
6. Copy CmdUI.PRF, run CopyCmdUIToAppData.bat
7. Open VS 2005, now the toolbars should be restored.
