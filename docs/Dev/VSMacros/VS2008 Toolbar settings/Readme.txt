Steps to restore the Toolbar settings.

1. Close all instances of IDEs.
2. Load the macros, used by toolbars.
	2.1 Merge the "VS 2008 LoadMacroProject RJK_InDn.reg"
3. Open VS 2008.
4. Import settings file "Exported-VS2008ToolbarSettings 2012-Feb-13.vssettings"
5. Close IDEs.
6. Copy CmdUI.PRF, run CopyCmdUIToAppData.bat
7. Open VS 2008, now the toolbars should be restored.
